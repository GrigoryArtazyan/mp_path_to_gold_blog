An interactive blog exploring Michael Phelps' Olympic career - the most decorated Olympian in history with 28 medals (23 gold, 3 silver, 2 bronze).

**Live Site:** [https://grigoryartazyan.github.io/mp_path_to_gold_blog/](https://grigoryartazyan.github.io/mp_path_to_gold_blog/)

## What's Inside

- Interactive visualizations of medal wins from 2004-2016
- Animated charts showing career progression
- Analysis of each Olympic Games performance
- Career statistics and highlights

## Built With

- Quarto (for the blog)
- R (for data visualization)
- plotly (for interactive charts)

## How to Run Locally

1. Clone this repo
2. Install R packages: `tidyverse`, `plotly`, `DT`
3. Run `quarto preview` in the terminal
4. View at http://localhost:XXXX

## Data Sources

- World Aquatics (FINA) Official Records
- International Olympic Committee
